Installing WSL

Installing WSL is straightforward on modern versions of Windows 10 and Windows 11. This guide focuses on WSL 2, which is the recommended version for most users.

Requirements

  • Windows 10 version 2004 or later (Build 19041 or higher), or any recent Windows 11 build.
  • Administrator privileges (you need to run some commands as an administrator).
  • Internet connection (for downloading the Linux distribution and optional components).

Quick installation with wsl --install

On supported systems, the simplest way to install WSL is by using a single command in an elevated PowerShell or Windows Terminal: 

wsl --install

This command will:

  • Enable the required Windows features.
  • Download and install the default Linux distribution (usually Ubuntu).
  • Set WSL 2 as the default version (on supported systems).

After the installation completes, you may need to restart your computer. When you start the distribution for the first time, you will be asked to create a Linux user name and password.

Manual installation steps (advanced)

If wsl --install is not available or if you want full control, you can enable WSL manually:

  1. Enable WSL and Virtual Machine Platform
    Open an elevated PowerShell and run: 
    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
  2. Restart Windows
    A reboot is required for the changes to take effect.
  3. Set WSL 2 as the default
    After reboot, open a terminal and run: 
    wsl --set-default-version 2
  4. Install a Linux distribution
    Open the Microsoft Store and search for distributions such as:
    • Ubuntu
    • Debian
    • Kali Linux
    • openSUSE Leap
    Choose one, click Get, then Install. When done, launch it from the Start menu and follow the initial setup.

Managing multiple distributions

You can install several Linux distributions side by side. Use these commands in PowerShell or Command Prompt: 

# List all installed distributions
wsl --list --verbose

# Set the default distribution
wsl --set-default <DistributionName>

# Set a distribution to use WSL 2
wsl --set-version <DistributionName> 2

Replace <DistributionName> with names like Ubuntu, Debian, etc., as shown by wsl --list --verbose.

Uninstalling and resetting WSL

If you want to remove a specific distribution, you can:

  1. Open Settings > Apps > Installed apps.
  2. Find the distribution (e.g., Ubuntu) and uninstall it.

To unregister (reset) a distribution from the command line: 

wsl --unregister <DistributionName>

Warning: Unregistering a distribution will delete all files and configuration inside that distribution. Make backups first.
