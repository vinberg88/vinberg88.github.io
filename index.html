Min Personliga WSL-guide

Välkommen till din snabbguide för Windows Subsystem for Linux (WSL). Här hittar du viktig information och användbara kommandon för att få ut det mesta av din Linux-miljö på Windows.

---

Vad är WSL?

Windows Subsystem for Linux (WSL) låter utvecklare köra en GNU/Linux-miljö — inklusive de flesta kommandoradsverktyg, verktyg och applikationer — direkt på Windows, omodifierad, utan omkostnaderna för en traditionell virtuell maskin eller dual-boot-konfiguration.

Det är ett kraftfullt verktyg för utvecklare som vill utnyttja Linux kraft i sin Windows-miljö, perfekt för webbutveckling, skriptning, systemadministration och mycket mer.

Installation och Grundläggande Användning

Installation (WSL 2 rekommenderas)

Öppna PowerShell eller Kommandotolken som administratör och kör:

wsl --install

Detta installerar WSL, Ubuntu (som standarddistribution) och de nödvändiga komponenterna. Starta om datorn efter installationen.

Starta WSL

Du kan starta din standard-Linux-distribution genom att:

  • Öppna "Ubuntu" (eller din distribution) från Startmenyn.
  • Öppna Kommandotolken eller PowerShell och skriva wsl.
---

Vanliga WSL-kommandon i Windows

Dessa kommandon körs i **Kommandotolken (CMD)** eller **PowerShell** i Windows, inte inuti din Linux-distribution.

  • wsl --list --verbose eller wsl -l -v: Visar en lista över installerade Linux-distributioner, deras status och vilken version av WSL de använder (WSL 1 eller WSL 2).
  • wsl --set-default <DistributionName>: Ställer in din standarddistribution. T.ex. wsl --set-default Ubuntu.
  • wsl --set-version <DistributionName> <Version>: Konverterar en distribution till WSL 1 eller WSL 2. T.ex. wsl --set-version Ubuntu 2.
  • wsl --shutdown: Stänger av alla körande WSL-distributioner. Användbart om du har problem eller vill frigöra resurser.
  • wsl --terminate <DistributionName>: Stänger av en specifik distribution. T.ex. wsl --terminate Ubuntu.
  • wsl --unregister <DistributionName>: Avinstallerar en distribution. **Varning: Detta raderar all data för den distributionen!**
  • wsl --update: Uppdaterar WSL-kärnan till den senaste versionen.

Tips och Tricks för WSL

Åtkomst till Windows-filer från Linux

Dina Windows-enheter är monterade under /mnt/ i Linux. T.ex. är din C-enhet tillgänglig via:

cd /mnt/c/Users/DittAnvändarnamn/Desktop

Åtkomst till Linux-filer från Windows

Du kan enkelt komma åt dina Linux-filer via nätverksplatsen \\wsl$ i Utforskaren. Skriv bara \\wsl$ i adressfältet i Utforskaren.

OBS: Undvik att redigera Linux-filer med Windows-verktyg om du inte använder en Linux-kompatibel editor som VS Code med WSL-extensionen, då det kan orsaka problem med filrättigheter och prestanda.

VS Code och WSL

Visual Studio Code har en fantastisk WSL-integration. Öppna din Linux-projektmapp i VS Code från WSL med:

code .

(Se till att du har VS Code installerat och den WSL Remote Development-extensionen installerad).
